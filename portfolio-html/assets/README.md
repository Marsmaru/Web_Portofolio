# 📁 Folder Assets — Petunjuk Penggantian Gambar

Taruh semua gambar kamu di folder ini, lalu pastikan nama filenya cocok dengan yang ada di `index.html`.

## Gambar yang dibutuhkan

| Nama File        | Dipakai di             | Ukuran ideal     |
|------------------|------------------------|------------------|
| `profile.jpg`    | Hero section & About   | 600 x 800 px     |

## Cara menambahkan gambar

1. Ambil foto terbaik kamu
2. Rename jadi `profile.jpg`
3. Taruh di folder `assets/` ini
4. Buka `index.html` di browser — foto akan langsung muncul

## Tips

- Gunakan format `.jpg` atau `.webp` agar ukuran file lebih kecil
- Ukuran file idealnya di bawah 500KB agar loading cepat
- Bisa compress di [squoosh.app](https://squoosh.app) (gratis, online)
- Foto portrait (tegak) lebih cocok untuk tampilan ini
